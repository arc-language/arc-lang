# Arc NVIDIA GPU Execution Model

This document explains how Arc's inline GPU execution works from end-to-end, using Arc's `extern` system to interface directly with NVIDIA's CUDA Driver API.

---

## The Complete Flow

```
Arc Source (inline GPU code)
    ↓
Arc Parser → AST
    ↓
Arc IR Generator → SSA IR
    ↓
PTX Backend → PTX Assembly (text)
    ↓
Arc Runtime (via extern cuda driver API)
    ↓
cuModuleLoadData(ptx_string)
    ↓
NVIDIA Driver JIT → SASS Machine Code
    ↓
GPU Execution
```

---

## Step-by-Step Breakdown

### Step 1: User Writes Inline GPU Code

```arc
// user_code.arc
func main() {
    let data = gpu.unified_malloc<float32>(1024)
    
    // Initialize on CPU
    for i in 0..1024 {
        data[i] = cast<float32>(i)
    }
    
    // Execute on GPU - inline kernel
    gpu func(arr: *float32, n: usize) {
        let idx = gpu.thread_id()
        if idx < n {
            arr[idx] = arr[idx] * 2.0
        }
    }(data, 1024)
    
    // Result is in data[0..1024]
    io.printf("Result: %f\n", data[0])
}
```

### Step 2: Compiler Detects GPU Kernel

When the Arc compiler sees:
```arc
gpu func(arr: *float32, n: usize) { ... }(data, 1024)
```

It recognizes this as:
1. **GPU execution context** (`gpu` keyword)
2. **Inline anonymous function** (the kernel)
3. **Immediate invocation** (args in parens)

The compiler extracts the kernel and compiles it separately:

```
Main Program Path:          GPU Kernel Path:
Arc → IR → AMD64            Arc → IR → PTX
```

### Step 3: PTX Generation

The compiler generates PTX assembly from the kernel:

```ptx
.version 7.0
.target sm_75
.address_size 64

.visible .entry __arc_gpu_kernel_1(
    .param .u64 arr,
    .param .u64 n
) {
    .reg .u32 %tid;
    .reg .u64 %r<4>;
    .reg .f32 %f<2>;
    .reg .pred %p1;
    
    // let idx = gpu.thread_id()
    mov.u32 %tid, %tid.x;
    
    // if idx < n
    ld.param.u64 %r1, [n];
    cvt.u64.u32 %r2, %tid;
    setp.lt.u64 %p1, %r2, %r1;
    @!%p1 bra EXIT;
    
    // Calculate address: arr + idx * 4
    mul.wide.u32 %r3, %tid, 4;
    ld.param.u64 %r4, [arr];
    add.u64 %r4, %r4, %r3;
    
    // Load arr[idx]
    ld.global.f32 %f1, [%r4];
    
    // Multiply by 2.0
    mul.f32 %f1, %f1, 2.0;
    
    // Store result
    st.global.f32 [%r4], %f1;
    
EXIT:
    ret;
}
```

### Step 4: PTX Embedded in Binary

The compiler embeds the PTX string into the Arc executable:

```arc
// Generated by compiler (conceptual)
const __gpu_kernel_1_ptx: string = ".version 7.0\n.target sm_75\n..."
```

### Step 5: Runtime Execution via CUDA Driver API

Arc interfaces with NVIDIA's CUDA Driver API using `extern`:

```arc
// lib/gpu/cuda.arc
extern cuda {
    // Initialize CUDA
    func cuInit "cuInit" (flags: uint32) int32
    
    // Device management
    func cuDeviceGet "cuDeviceGet" (device: *int32, ordinal: int32) int32
    func cuCtxCreate "cuCtxCreate" (ctx: **void, flags: uint32, device: int32) int32
    
    // Module loading (PTX → GPU)
    func cuModuleLoadData "cuModuleLoadData" (module: **void, ptx: *byte) int32
    func cuModuleGetFunction "cuModuleGetFunction" (func: **void, module: *void, name: *byte) int32
    
    // Kernel execution
    func cuLaunchKernel "cuLaunchKernel" (
        func: *void,

        sharedMemBytes: uint32,

        extra: **void
    ) int32
    
    // Memory management
    func cuMemAlloc "cuMemAlloc" (ptr: **void, size: usize) int32
    func cuMemcpyHtoD "cuMemcpyHtoD" (dst: *void, src: *void, size: usize) int32
    func cuMemcpyDtoH "cuMemcpyDtoH" (dst: *void, src: *void, size: usize) int32
    func cuMemFree "cuMemFree" (ptr: *void) int32
}
```

## Complete Runtime Implementation (Pure Arc)

```arc
// lib/gpu/runtime.arc
namespace gpu

struct CUDAContext {
    device: int32
    context: *void
    initialized: bool
}

let ctx: CUDAContext = CUDAContext{device: 0, context: null, initialized: false}

func init() {
    if ctx.initialized {
        return
    }
    
    // Initialize CUDA
    let result = cuda.cuInit(0)
    if result != 0 {
        raise("CUDA initialization failed")
    }
    
    // Get device
    result = cuda.cuDeviceGet(&ctx.device, 0)
    if result != 0 {
        raise("Failed to get CUDA device")
    }
    
    // Create context
    result = cuda.cuCtxCreate(&ctx.context, 0, ctx.device)
    if result != 0 {
        raise("Failed to create CUDA context")
    }
    
    ctx.initialized = true
}

struct GPUKernel {
    module: *void
    function: *void
    name: string
}

func load_ptx(ptx_code: string, kernel_name: string) GPUKernel {
    init()  // Ensure CUDA is initialized
    
    let module: *void = null
    
    // Load PTX module
    let result = cuda.cuModuleLoadData(&module, ptx_code)
    if result != 0 {
        raise("Failed to load PTX module")
    }
    
    // Get kernel function
    let function: *void = null
    result = cuda.cuModuleGetFunction(&function, module, kernel_name)
    if result != 0 {
        raise("Failed to get kernel function")
    }
    
    return GPUKernel{
        module: module,
        function: function,
        name: kernel_name
    }
}

func launch(kernel: GPUKernel, grid_dim: uint32, block_dim: uint32, args: **void) {
    let result = cuda.cuLaunchKernel(
        kernel.function,
        grid_dim, 1, 1,      // Grid dimensions
        block_dim, 1, 1,     // Block dimensions
        0,                   // Shared memory
        null,                // Stream
        args,                // Kernel arguments
        null                 // Extra
    )
    
    if result != 0 {
        raise("Kernel launch failed")
    }
}

// Unified memory allocation (simplest API)
func unified_malloc<T>(count: usize) *T {
    let size = count * sizeof<T>
    let ptr: *void = null
    
    let result = cuda.cuMemAllocManaged(&ptr, size, 1)  // 1 = CU_MEM_ATTACH_GLOBAL
    if result != 0 {
        raise("Unified memory allocation failed")
    }
    
    return cast<*T>(ptr)
}

func unified_free(ptr: *void) {
    cuda.cuMemFree(ptr)
}

// Special function for getting thread ID (compiler intrinsic)
func thread_id() usize {
    // This is replaced by the compiler with PTX: mov.u32 %tid, %tid.x
    // In the kernel context only
    return 0  // Placeholder
}
```

## How the Compiler Transforms GPU Code

### Original Arc Code:
```arc
gpu func(arr: *float32, n: usize) {
    let idx = gpu.thread_id()
    if idx < n {
        arr[idx] = arr[idx] * 2.0
    }
}(data, 1024)
```

### Compiler Transformation:

**1. Extract kernel code**
```arc
func __arc_gpu_kernel_1(arr: *float32, n: usize) {
    let idx = gpu.thread_id()
    if idx < n {
        arr[idx] = arr[idx] * 2.0
    }
}
```

**2. Generate PTX from kernel**
```arc
const __arc_gpu_kernel_1_ptx: string = "
.version 7.0
.target sm_75
.address_size 64
.visible .entry __arc_gpu_kernel_1(
    .param .u64 arr,
    .param .u64 n
) {
    // ... PTX assembly ...
    ret;
}
"
```

**3. Replace inline GPU call with runtime call**
```arc
// Original:
gpu func(arr: *float32, n: usize) { ... }(data, 1024)

// Becomes:
{
    let kernel = gpu.load_ptx(__arc_gpu_kernel_1_ptx, "__arc_gpu_kernel_1")
    let args: array<*void, 2> = {
        cast<*void>(&data),
        cast<*void>(&1024)
    }
    gpu.launch(kernel, 4, 256, &args[0])  // 4 blocks × 256 threads = 1024 threads
}
```

## Memory Management Examples

### Unified Memory (Simplest)
```arc
func example_unified() {
    // Allocate memory accessible from both CPU and GPU
    let data = gpu.unified_malloc<float32>(1024)
    
    // Write from CPU
    for i in 0..1024 {
        data[i] = cast<float32>(i)
    }
    
    // Read/write from GPU
    gpu func(arr: *float32, n: usize) {
        let idx = gpu.thread_id()
        arr[idx] = arr[idx] * 2.0
    }(data, 1024)
    
    // Read from CPU (automatically synced)
    let result = data[0]
    
    gpu.unified_free(data)
}
```

### Explicit Memory Management (Performance)
```arc
func example_explicit() {
    // CPU memory
    let cpu_data = alloca(float32, 1024)
    for i in 0..1024 {
        cpu_data[i] = cast<float32>(i)
    }
    
    // GPU memory
    let gpu_data: *float32 = null
    cuda.cuMemAlloc(cast<**void>(&gpu_data), 1024 * sizeof<float32>)
    
    // Copy CPU → GPU
    cuda.cuMemcpyHtoD(gpu_data, cpu_data, 1024 * sizeof<float32>)
    
    // Execute kernel
    gpu func(arr: *float32, n: usize) {
        let idx = gpu.thread_id()
        arr[idx] = arr[idx] * 2.0
    }(gpu_data, 1024)
    
    // Copy GPU → CPU
    cuda.cuMemcpyDtoH(cpu_data, gpu_data, 1024 * sizeof<float32>)
    
    // Cleanup
    cuda.cuMemFree(gpu_data)
    
    let result = cpu_data[0]
}
```

## Grid and Block Dimension Calculation

The compiler needs to determine how many threads to launch:

```arc
// Compiler calculates grid/block dimensions
func calculate_grid_dims(total_threads: usize) (uint32, uint32) {
    const BLOCK_SIZE: uint32 = 256  // Threads per block
    
    let blocks = (total_threads + BLOCK_SIZE - 1) / BLOCK_SIZE
    
    return (cast<uint32>(blocks), BLOCK_SIZE)
}

// Example: 1024 threads
// blocks = (1024 + 255) / 256 = 4
// Grid: 4 blocks × 256 threads/block = 1024 threads
```

For the inline kernel:
```arc
gpu func(arr: *float32, n: usize) { ... }(data, 1024)
```

The compiler knows:
- `n = 1024` (the size parameter)
- Each thread processes one element
- Launch `ceil(1024 / 256) = 4` blocks of 256 threads

## Complete Example: Matrix Multiply

```arc
func matmul_gpu(A: *float32, B: *float32, C: *float32, N: usize) {
    let total_elements = N * N
    
    gpu func(a: *float32, b: *float32, c: *float32, n: usize) {
        let idx = gpu.thread_id()
        let row = idx / n
        let col = idx % n
        
        if row < n && col < n {
            let mut sum: float32 = 0.0
            for k in 0..n {
                sum += a[row * n + k] * b[k * n + col]
            }
            c[row * n + col] = sum
        }
    }(A, B, C, N)
}

func main() {
    const N: usize = 1024
    
    // Allocate unified memory
    let A = gpu.unified_malloc<float32>(N * N)
    let B = gpu.unified_malloc<float32>(N * N)
    let C = gpu.unified_malloc<float32>(N * N)
    
    // Initialize matrices on CPU
    for i in 0..(N * N) {
        A[i] = 1.0
        B[i] = 2.0
    }
    
    // Compute on GPU
    matmul_gpu(A, B, C, N)
    
    // Result is in C
    io.printf("C[0,0] = %f\n", C[0])
    
    // Cleanup
    gpu.unified_free(A)
    gpu.unified_free(B)
    gpu.unified_free(C)
}
```

## Compiler Internals: PTX Code Generation

When the compiler encounters `gpu.thread_id()`:

```arc
// Arc IR
%tid = call @gpu.thread_id()

// PTX Generation
mov.u32 %tid, %tid.x
```

When the compiler encounters array access:

```arc
// Arc code
arr[idx] = arr[idx] * 2.0

// Arc IR
%1 = mul %idx, 4              // sizeof(float32) = 4
%2 = add %arr_base, %1        // Calculate address
%3 = load %2                  // Load value
%4 = fmul %3, 2.0             // Multiply
store %4, %2                  // Store back

// PTX
mul.wide.u32 %r1, %tid, 4
add.u64 %r2, %arr_base, %r1
ld.global.f32 %f1, [%r2]
mul.f32 %f1, %f1, 2.0
st.global.f32 [%r2], %f1
```

## Key Points

1. **No Go runtime needed** - Arc uses `extern` to call CUDA Driver API directly
2. **PTX is embedded** - Compiler embeds PTX strings in the Arc binary
3. **JIT happens at runtime** - First kernel launch compiles PTX → SASS (cached)
4. **Unified memory is easiest** - CPU and GPU share the same pointers
5. **Grid dimensions are calculated** - Compiler determines blocks/threads from array size

## The Magic Moment

When you write:
```arc
gpu func(data: *float32, n: usize) {
    data[gpu.thread_id()] *= 2.0
}(array, 1024)
```

**What actually happens:**

1. Compiler extracts kernel → generates PTX
2. PTX embedded in binary as string constant
3. At runtime: `cuModuleLoadData(ptx_string)` loads to GPU
4. NVIDIA driver JIT compiles PTX → SASS (once, then cached)
5. `cuLaunchKernel()` executes 1024 threads in parallel
6. Each thread runs the kernel with a different `thread_id()`
7. GPU completes, results are in memory

**All from one line of inline Arc code.** That's the power of native GPU integration. 🚀
